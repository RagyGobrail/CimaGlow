# CimaGlow
🎬 Movie Explorer – AI-Powered Movie Discovery App

Description:

Movie Explorer is a responsive web application that allows users to search, explore, and discover movies and TV shows using real-time data from the TMDB API. The platform provides detailed movie information including ratings, posters, genres, and descriptions with a clean and modern UI.

Key Features:

Real-time movie search using TMDB API
Responsive design for mobile and desktop
Dark mode / modern UI (if you have it)
Movie details page (rating, overview, genres)
API integration with dynamic data fetching
Fast and optimized React frontend

Tech Stack:
React.js, TypeScript, CSS, TMDB API, Vercel
